String

注意

在程序中，所有双引号字符串都是String类的对象，没有new也是

特点

  • 字符串的内容永不发生改变，是一个常量，在创建之后不能更改
  • 字符串可以共享使用
  • 效果上相当于char[]型数组，底层原理是byte字节数组
  • 字符串缓冲区支持可变字符串

创建字符串方式

  1. public String()创建空白字符串
  2. public String(char[] array)根据字符数组内容创建字符串
  3. public String(byte[] array)根据字节数组创建字符串

字符串常量池

在定义字符串的时候，使用双引号直接创建的时候，就在字符串常量池中

对于==符号来说，对于基本数据类型来说是比较数值，对于引用类型比较地址

字符串常量池被放在堆内存中，

字符串常见方法

public int length();获取字符串长度

public String concat(String str);字符串拼接

public char charAt(int index);获取索引处单个字符

public int indexOf(String str);查找指定参数字符串第一次出现的位置，没有返回-1

public String substring(int index);从字符串索引位置一直到字符串结尾

public String substring(int begin,int end);截取[begin,end)位置字符串

public char[] toCharArray();将当前字符串作为char[] 数组返回

public byte[] getBytes();获得当前字符串字节数据

public String replace(CharSequence oldString,CharSequence newString);将所有老字符串替换成为新字符串，返回替换后的新字符串，这里的CharSequence是一个接口
